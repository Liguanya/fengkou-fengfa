# 🌀 风口风阀清单定额自动编制系统

基于浏览器端的风口风阀工程量清单定额自动编制工具，支持上传Excel文件，自动计算周长、面积，套用定额，计算主材价格。

## 功能特点

- 📊 **自动识别产品类型**：支持 70度防火阀、280度排烟防火阀、蝶阀、止回阀、百叶风口、散流器等 32 类产品
- 📐 **智能尺寸提取**：自动从名称中提取尺寸（支持 `400*600` 和 `φ400` 格式）
- 📈 **自动计算**：计算周长、面积，套用对应定额
- 💰 **主材价计算**：根据公式自动计算主材价格
- 🔒 **隐私安全**：所有数据处理在浏览器本地完成，不会上传到任何服务器
- 📥 **一键导出**：支持导出处理后的 Excel 文件

## 主材价计算公式

### 防火阀类
- 70度防火阀（面积<0.5㎡）：`(G*1480+220)*0.5`
- 70度防火阀（0.5㎡≤面积<1㎡）：`(G*1418+260)*0.5`
- 70度防火阀（面积≥1㎡）：`(G*1380+280)*0.5`
- 280度排烟防火阀：根据面积分档计算

### 风口类
- 防雨百叶风口：`(G*720+840)*0.5`
- 双层百叶风口：`(G*720+72)*0.5`
- 单层百叶风口：`(G*660+66)*0.5`
- 散流器：`(G*1020+96)*0.5`

更多公式详见系统内置的 32 类产品主材价计算规则。

## 使用方法

1. 打开网页，上传包含风口、风阀工程量清单的 Excel 文件
2. 系统自动识别产品类型、提取尺寸、计算定额和价格
3. 查看处理结果，确认无误后点击"导出Excel"下载
